## Data Safety / Privacy Declarations

### Data Collected

| Data Type | Collected | Shared | Purpose |
|-----------|-----------|--------|---------|
| Email address | Yes | No | Account creation, authentication |
| Name | Yes | No | Profile display, community |
| Profile photo | Optional | Yes* | Community, stylist directory |
| Location | Optional | Yes* | Stylist directory (public listing) |
| Payment info | Yes | No | Processed by Stripe (not stored) |
| Usage data | Yes | No | Analytics, app improvement |
| Device info | Yes | No | Push notifications, crash reporting |

*Shared only when user opts into public stylist directory

### Data Handling

- **Encryption:** All data encrypted in transit (HTTPS/TLS)
- **Account deletion:** Users can request account deletion via support email
- **Data retention:** Data retained while account is active
- **Third-party services:** Stripe (payments), Supabase (backend), Mux (video)
