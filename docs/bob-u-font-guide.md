# BOB CO · FONT GUIDE
## Primary Heading Font
### CLOCKS
- Use for all H1 and major headlines
- Editorial, high-contrast serif
- Intended for large display sizes only
- Fallback: Playfair Display
- Use when CLOCKS is unavailable (web-safe / Canva-safe). Sometimes Clocks doesn’t look quite right and then you can use Playfair in it’s place.

## Body & Supporting Text
### DM Sans
- Use for all:
- Subheadings (H2–H4)
- Paragraph/body text
- Navigation
- Buttons & CTAs
- Weights:
- Subheads: Medium or SemiBold
- Body text: Regular
- Buttons: Medium (optional ALL CAPS)
- Clean, modern, highly readable; balances the editorial serif headings

## Accent / Script Font
### Brittany
- Use sparingly for emphasis only:
- Accent words in headlines
- Callout phrases
- Short stylistic moments
- Do not use for long text or paragraphs
- Visual Hierarchy (Intent)
- Headlines = elegant + editorial
- Body text = clean + approachable
- Script = expressive, used intentionally
